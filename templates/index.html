Help / getting started

  1. Count the number of non-land cards you have
  2. Decide how much lands you want to add in the deck
    • For a 40 cards deck, it's advised to put 17 lands and 23 non-lands
  3. For each color, sum the colored mana on your cards' costs
    • For example, a 3️⃣☀️☀️ count as 2 white mana, a 3️⃣☀️ count as one
  4. Enter all these values in the boxes and click "Compute!"

About

godraft is a simple webapp to help players during Magic the Gathering(tm) drafts. First version features a tool to compute how much lands of each colors you should add to your draft deck.

godraft is open source. Source code can be found here. Issues/suggestions can be submitted there.
